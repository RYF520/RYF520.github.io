nodejs和vue脚手架

2023/6/4 前端 node.js
我们在使用vue脚手架之前需要一些准备，例如安装node.js（简单来说Node.js 是一个开源、跨平台的 JavaScript 运行时环境。）
  1. 下载nodejs安装包
    官网地址Node.js
    点击下载此版本
  1. 下载后进行安装
  • 点击下一步
  • 如果此处出现错误则按下一步操作

  • 此处不要勾选，然后一路下一步安装即可
  1. 查看是否安装成功
    按住win+R，输入cmd
  • 输入node -v 和 npm -v 出现以下版本说明安装成功
  1. 设置为国内镜像输入 npm config set registry https://registry.npmmirror.com 此时没有反应属于正常
  2. 安装vue脚手架
    输入 npm install @vue/cli -g
  3. 安装完成进行查看
    输入 vue -V 出现以下版本号即为安装成功
